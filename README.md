# 学生管理系统

## 简介：

题目虽然是学生管理系统，但功能包含(学生，教师，管理员),项目基于springboot2.1.x实现的管理系统。如果你是spring boot初学者，那么此项目非常适合你。如果喜欢，请随手给个star，谢谢。

## 编译环境 ：

>jdk 1.8 
>
>mysql 5.5 
>
>tomcat 7

## 框架 ：

>springboot2.0 
>
>mybatis

## jar包管理工具：

> Maven

## 编译器 ：

>IDEA

## 完成进度：

	> 完成

## 系统功能 :

> 学生信息管理
>
> 班级信息管理
>
> 教师信息管理
>
> 课程信息管理
>
> 选课信息管理
>
> 考勤信息管理
>
> 请假信息管理
>
> 成绩信息管理
>
> 系统管理
